- [**TransferNet**](https://www.saliltimalsina.com.np/transfernet) — Mobile money transfer app for the Australian diaspora. KYC, live rates, five-stage tracking.
- [**Jobbicus**](https://www.saliltimalsina.com.np/jobbicus) — Career platform for job seekers and employers. Profile, feed, postings, proposals, courses.
- [**Skathi**](https://www.saliltimalsina.com.np/skathi) — Music platform for South Asian listeners. iOS streaming + web with beats marketplace, artist booking, events, merch.
